## API Usage Examples

### Starting a session:
```
POST /chat/start
Response: {"session_id": "uuid-here", "message": "New chat session started"}
```

### Sending messages:
```
POST /chat/message
{
  "message": "I have fever and headache", 
  "session_id": "uuid-here"
}
Response: {
  "success": true,
  "response": "Bot response here",
  "image_path": "category_dist.png",
  "language": "english",
  "session_id": "uuid-here"
}
```

### Getting images:
```
GET /chat/image/category_dist.png
Response: [PNG Image File]
```

### Generating reports:
```
POST /report/generate/uuid-here
{
  "name": "John Smith",
  "age": 28,
  "gender": "male", 
  "location": "New York"
}
Response: {
  "success": true,
  "download_url": "/report/download/John_Smith_DoctorAI_Report.pdf"
}
```
